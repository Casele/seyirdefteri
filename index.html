Firebase Kurulumu Gerekli
HTML dosyasındaki FIREBASE_CONFIG değerlerini doldurmadın. Aşağıdaki adımları izle:
1
console.firebase.google.com adresine git ve Proje Oluştur'a tıkla. Bir isim ver (örn. "cinemate"), Google Analytics'i kapat ve oluştur.
2
Sol menüde Realtime Database'e tıkla → Veritabanı Oluştur → Lokasyon seç (herhangi biri olur) → Test modunda başla seç → Etkinleştir.
3
Sol üstteki ⚙️ Proje Ayarları → Web Uygulaması Ekle (</> ikonu) → İsim ver → Kaydet. Sana bir firebaseConfig nesnesi verecek. Onu kopyala.
4
Bu HTML dosyasını bir metin editörüyle aç (Notepad, VSCode, vb.) ve en üstteki FIREBASE_CONFIG nesnesindeki değerleri kopyaladıklarınla değiştir. Sonra dosyayı app.netlify.com/drop'a sürükle-bırak.
Örnek yapı:
const FIREBASE_CONFIG = {
  apiKey: "AIzaSy...",
  authDomain: "proje.firebaseapp.com",
  databaseURL: "https://proje-default-rtdb.firebaseio.com",
  projectId: "proje",
  ...
};
